# Task7 固定双读图融合（F2）预注册

日期：2026-07-12
任务边界：100% 覆盖、固定阈值 0.5 的图像模型；无拒识、人工复核、行为路由或置信度特征。

## 1. 唯一融合规则

F2 只包含两个独立图像成员：

1. 锁定 C1：SigLIP-L@512 六视图 dense-token gated 模型；
2. F1 主变体：六视图加三个无标签随机局部袋、共享视觉头与跨袋一致性。

病例概率固定为：

`p(F2) = 0.5 × p(C1) + 0.5 × p(F1)`

阈值固定为 `0.5`。不学习权重，不扫描阈值，不根据成员置信度选择预测，不输入来源、批次、难度、错误或拒识状态。

## 2. 机制问题

F1 在 source-LODO 相对其同实验基础组提高 BAcc `0.0232`、AUC `0.0460`，并在 B2/B3/TC 上正向，但相对该基础组损害 B1。F2 检验两个真正读图的模型是否有固定互补性：C1 提供稳定全局/大区域证据，F1 提供位置不变的局部证据。

这不是“第二阶段猜第一阶段是否正确”，也不是按置信度纠偏。若固定平均不能同时改善 OOF、LODO 和边界亚型，则不再搜索融合权重。

## 3. 评估与比较

- 使用已经锁定且病例级对齐的五折 OOF 与 source-LODO 概率；
- 报告 C1、已锁定图像融合 C2、F1、F2；
- 固定阈值 0.5；
- BAcc、AUC、敏感度、特异度、三来源、六亚型；
- 5,000 次来源×标签分层病例 bootstrap；
- 所有运算均在服务器完成，不导出患者级预测。

## 4. 晋级门槛

F2 只有全部满足才 PASS：

1. 相对 C1，OOF 与 source-LODO BAcc 均至少 `+0.010`；
2. OOF 与 LODO 均优于 F1 单独模型；
3. LODO 至少 2/3 来源正增益，任一来源不得低于 `-0.020`；
4. B1、B2 的 LODO 准确率相对 C1 均不得下降超过 `0.020`；
5. OOF 不得低于已锁定 C2 超过 `0.005`；
6. LODO BAcc 必须超过已锁定 C2。

PASS 只允许把 F2 保留为“待全新盲法外部验证的内部候选”，不等于外部泛化已解决。NO-GO 后不得追加权重、阈值或成员搜索。
