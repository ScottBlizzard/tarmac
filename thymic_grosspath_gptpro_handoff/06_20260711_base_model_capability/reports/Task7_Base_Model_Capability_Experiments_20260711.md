# Task7 纯图像基础能力扩展实验总结（内部锁定完成）

日期：2026-07-11

## 1. 固定主线

- 任务：胸腺大体图 Task7 二分类，低危 `A/AB/B1` 对高危 `B2/B3/TC`。
- 首要目标：提升 100% 覆盖、无拒识、无人工复核条件下的纯图像基础分类能力及跨域泛化。
- 工作流、拒识和二阶段复核仅作为后续安全层，不作为基础模型能力的替代指标。
- 内部开发：old_data 285 例 + third_batch 306 例，共 591 例。
- 历史严格外部：108 例；新外部注册表：162 例。
- 本轮所有模型选择、阈值与融合配方只使用内部数据。外部评估必须等内部候选清单锁定后才能启动。

## 2. 评估协议

- 使用锁定的五折病例级切分；发现四域注册表中 247 个旧 fold 值与权威切分冲突，统一以权威 `split.csv` 覆盖。
- 五折训练中，每折使用一个测试折、下一个折作为早停验证折，其余三折训练。
- 主指标：固定阈值 0.5 下的 BAcc、AUC、敏感度、特异度、FN/FP；同时检查三个来源的最低 BAcc、B1/B2 亚型表现和折间稳定性。
- 另增来源整批留出（source-LODO）：依次把 batch1、batch2、third_batch 整批排除训练后再测试，用作内部伪外部泛化证据。
- 最终等权融合同时参考普通五折 OOF 和 source-LODO；融合成员锁定后才允许提取外部 token。

## 5. 医生概念与错误机制

医生肉眼所见表仅在训练期提供可观察概念辅助或错误分析，推理时不读取文本、病例号或诊断。

当前错误富集提示：

- 高危假阴性常呈低危样大体表型：质软、结节/分叶、包膜完整或可见。
- 低危假阳性常呈高危样表型：灰红、边界不清、囊性变、肺/胸膜粘连、包膜缺失。
- 因而 hard 不是单一难度层级，而是病理真值与大体视觉表型冲突、ROI 不足、质量问题和域偏移的混合。

已生成医生人工复核候选表，但尚未把候选分析写成“医生已确认”的结论。

现有自动质量表只能与 Task7 的 batch1 117 例匹配，不能代表全部 591 例。该子集内 `fair` 组准确率 0.8421，反而高于 `good` 组 0.7041；质量分最高分层误差率也更高，主要受亚型构成混杂。因此当前没有证据把低质量认定为主要错误驱动，图像增强只作为保留原图的软门控对照，而不是强制预处理。

## 6. 已证实的负结果

- 普通均值池化显著弱于 dense gated attention。
- REx、GroupDRO 当前参数下未超过普通 qkvb gated。
- ConvNeXt-B 即使使用 gated token pooling 也明显弱。
- 简单 Gray-world 白平衡没有提升基础 BAcc。
- 原始 B1/B2 hard-pair 专家没有稳定改善边界病例。
- 多 ROI、DANN 和 ROI-GroupDRO 的增益至多约 1 个百分点，尚未解决跨来源不稳。
- 概念、亚型和边界的重联合监督出现明显负迁移，医生概念更适合轻量辅助、冲突建模和错误分层。
- background-only 审计接近随机，未发现仅靠背景即可完成分类的强捷径。
- 首个 qkv/proj LoRA 的 AUC 提高到 0.7841，但固定阈值 BAcc 仍为 0.7073，不能只以排序指标判断成功。
- LoRA 去类别权重后敏感度降至 0.5650，确认总体准确率或特异度提高不能替代类别平衡。
- LoRA 从注意力投影扩大到 MLP 后 BAcc 降至 0.6820，当前瓶颈不是适配参数数量不足。
- 内部 VICReg 预适配未改善最终 OOF，当前小数据自监督配方不足以弥合来源差异。
- 纯六分类派生风险没有改善 Task7；六分类 Accuracy 0.5178、宏 BAcc 0.3985，B1/B2 风险准确率仅 0.4194/0.5506。
- 二分类/六分类 50:50 融合进一步降至 0.6798 BAcc，六分类概率不能作为强制融合分支。
- 加入医生概念后层级融合仍只有 0.7034 BAcc，六分类当前仅适合轻辅助和错误分层。
- B1/B2 relevance gate 的 B1/B2 风险准确率为 0.3871/0.6180，未形成边界能力提升。
- 扩大为低危/B2 专家后 third_batch 敏感度降至 0.5488，边界专家路线未改善跨来源泛化。
- gate 监督/平衡 MoE 的 third_batch 敏感度仅 0.5732，B2 风险准确率 0.4944，强制专家分工造成负迁移。
- MoE 加 GroupDRO 后最低来源仍仅 0.6310、总体 BAcc 0.6852；当前最佳仍是自由门控 run 211。
- crop+四象限的局部视图平均权重均非零，但总体 BAcc 0.6989、B1 风险准确率 0.3710，局部 token 未转化为稳定边界能力。
- 同一五视图表示改用自由 MoE 后单模型 BAcc 提高到 0.7208，并把 208+219+255 融合推到 0.7622；该信号必须由 source-LODO 和锁定外部复核。
- whole/ROI 强一致性把 third_batch 高危召回压到 0.5000，说明两种视图不能被强制学成同一判断。
- 序数亚型正则将 BAcc 降至 0.6722，A/B1 判对仅 0.4773/0.4516，不应把六亚型强制映射成连续视觉序列。
- AIMv2 两个独立种子均复现约 0.72 BAcc，但三 seed 平均仅 0.7271，异构融合优于简单 seed ensemble。
- focal loss 在 AIMv2 上将 BAcc 小幅提到 0.7278，但在 qkvb 上降至 0.6793 并导致 third 高危敏感度仅 0.4878，不是可直接推广的通用配方。
- 本地缓存的 DINOv2-L 即使使用 dense patch token，gated/view-gated 也只有 0.6761/0.6978 BAcc，且 batch2 下限偏低；换基底必须通过来源稳定性验证，不能只依赖预训练模型名称。
- 将自动对比度和轻度反锐化作为软门控视图仍将 third 高危敏感度降至 0.4390，因此低质量预处理不能无条件并入主模型。
- 医生概念规则标记的 359 个视觉-病理冲突病例确实比 232 个非冲突病例难；baseline 207 的两组 BAcc 为 0.6552/0.7549。但 10% 冲突标签软化将冲突组 BAcc 降到 0.6430，还低于自由 MoE 211 的 0.6857；医生概念适合用于分层和提出机制假设，不等于直接修改标签就有效。
- 类条件跨域对齐在普通头和 MoE 上均未改善最差来源，说明当前域差异中包含与诊断有关的信号，不能作统一类内收缩。
- 高危漏诊 sentinel 无论单独还是与 MoE 联合都未提高敏感度或 BAcc，额外二分支路没有学出与主头独立的高危证据。
- MixStyle 对 qkvb 单模型最差来源无增益，对 AIMv2 将总体 BAcc 提到 0.7230 但牺牲 third 高危召回；它的主要价值暂时是与高分辨率 MoE 的错误互补，必须由 source-LODO 验证。
- qkvb 强域增强 LoRA 的 AUC 为 0.7988，但 BAcc/敏感度仅 0.6909/0.5964；AIMv2 轻风格 LoRA 达到 0.7256 BAcc 和 0.8127 AUC，说明端到端轻适配对合适骨干有小幅价值，但仍没有达到将单模型提升到 0.75+ BAcc 的程度。
- 既往手工 ROI/颜色/纹理统计单独外部 BAcc 约 0.57，当前不重复作为主模型。

## 7. 多图病例可行性审计

- 591 例内部开发集中只有 17 例保留两张原始照片，占 2.9%；其余 574 例均为单图。

## 19. 临时候选的医生概念错误分层
