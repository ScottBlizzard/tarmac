## 3. D1 标签诱导 attention ROI

| 模型 | OOF BAcc/AUC | LODO BAcc/AUC |

| 同实验基础组 | 0.7519/0.8177 | 0.7125/0.7638 |

## 4. E1 无标签解剖 ROI

| 模型 | OOF BAcc/AUC | LODO BAcc/AUC |
| --- | ---: | ---: |
| 同实验基础组 | 0.7519/0.8177 | 0.7125/0.7638 |
| 解剖 ROI | 0.7302/0.7800 | 0.7051/0.7759 |
| 匹配随机 ROI | 0.7407/0.8027 | 0.7284/0.8004 |

解剖 ROI 相对基础组 OOF `-0.0217`、LODO `-0.0074`，并在两种协议下都不如随机 ROI，正式判定 NO-GO。随机 ROI 的 LODO 相对基础组 BAcc `+0.0158`（95% CI `[-0.0116,0.0446]`）、AUC `+0.0366`（95% CI `[0.0123,0.0617]`），但 OOF BAcc `-0.0111`，且来源方向不一致，不能作为现成提升结论。

E1 计划质控：591 例；随机 fallback 率 `3.55%`；组织覆盖率绝对差均值/P95 `0.0090/0.0206`；中心最小距离均大于 `0.20`。

## 5. 当前结论

1. 更大骨干、标签 attention 和固定解剖启发式均未提高稳定的全量直接分类能力。
2. D1 的域内提升在 LODO 反转，验证了“看起来更细”不等于学到可迁移形态。
3. E1 随机局部袋仅提供一个需严格验证的机制信号：局部重读提高 LODO 排序能力，但单袋方差和来源不稳定仍明显。
4. 下一步仅做预注册 F1 多随机袋共享头与跨袋一致性；若仍失败，不再围绕同一单图随机裁剪调袋数、权重或阈值。

## 6. F1 多随机袋一致性

| 模型 | OOF BAcc/AUC | LODO BAcc/AUC |
| --- | ---: | ---: |
| E1 同实验基础组 | 0.7519/0.8177 | 0.7125/0.7638 |
| E1 单随机袋 | 0.7407/0.8027 | 0.7284/0.8004 |
| 三袋均值 | 0.7352/0.8132 | 0.7358/0.8016 |
| 三袋一致性主变体 | 0.7450/0.8173 | 0.7357/0.8098 |

主变体相对 E1 基础组：OOF BAcc `-0.0069`，LODO BAcc `+0.0232`，LODO AUC `+0.0460`（95% CI `[0.0196,0.0742]`）。三来源 LODO BAcc 方向为 `+0.0425 / 0 / +0.0100`，但 B1 相对基础组下降 `0.1452`。因此 F1 正式 NO-GO：位置不变局部重读提高跨来源排序与部分高危亚型，却没有同时保住 B1 和域内强制分类。

## 7. F2 固定 C1+F1 双读图融合

F2 仅做一次固定 `0.5×C1 + 0.5×F1` 概率平均，不学习权重、不看置信度、不调阈值。

| 模型 | OOF BAcc/AUC | LODO BAcc/AUC |
| --- | ---: | ---: |
| C1 | 0.7477/0.8240 | 0.7397/0.8072 |
| 已锁定 C2 | 0.7514/0.8377 | 0.7441/0.8108 |
| F2 | 0.7401/0.8369 | 0.7443/0.8326 |

F2 相对 C1 的 OOF/LODO BAcc 为 `-0.0075/+0.0047`，AUC 为 `+0.0129/+0.0254`；LODO 第三批来源下降 `0.0480`，B2/B3 分别下降 `0.0449/0.1250`。虽然排序改善且 LODO AUC 高于 C2，固定 0.5 强制分类没有稳定提高，正式 NO-GO，不追加权重或阈值搜索。

## 8. 自动单图路线停止边界

当前证据已覆盖普通骨干扩容、原生细节网格、标签 attention、无标签解剖 ROI、随机局部袋、跨袋一致性和一次固定双读图融合。正向信号主要停留在 AUC/排序，B1/B2 与来源间错误搬移仍无法消除。在同一 591 例和同一张原始图上继续搜索结构、袋数、损失、融合权重或阈值，已经属于新的元选择层，而不是可信能力提升。

下一条有效信息路线是双读者盲法医生 ROI oracle。两个独立表单、无标签校验锁、匹配随机 ROI、五折/LODO 和 GO/NO-GO 分析器均已工程验证。若人工 ROI oracle 失败，应停止当前单图 ROI 开发，转向标准化切面近照、包膜/交界近照和全新多中心病例。
