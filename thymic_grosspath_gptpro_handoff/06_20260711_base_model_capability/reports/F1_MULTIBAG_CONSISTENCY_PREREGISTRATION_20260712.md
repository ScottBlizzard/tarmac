# Task7 多随机袋一致性（F1）预注册

日期：2026-07-12
任务边界：100% 覆盖、阈值固定 0.5 的直接图像二分类；无拒识、人工复核、置信度路由或行为特征。

## 1. 机制依据

E1 的解剖 ROI 未通过：相对同实验基础组，五折 BAcc `-0.0217`，source-LODO `-0.0074`，且 OOF/LODO 均不优于匹配随机 ROI。不能继续调整解剖分数。

E1 的匹配随机 ROI 在 source-LODO 相对基础组 BAcc `+0.0158`、AUC `+0.0366`，但五折 BAcc `-0.0111`，来源方向也不一致。F1 只检验一个后续假设：**单个随机局部袋方差过大；共享视觉头读取多个独立局部袋并学习位置不变性，能否把局部证据转化为稳定的直接分类增益。**

## 2. 固定输入与模型

- 锁定 C1 六个全局/大区域视图；
- 每例固定 3 个独立随机 ROI 袋；
- 每袋含一个高组织覆盖率中心和一个交界覆盖率中心，每个中心读取 `0.40`、`0.25` 两尺度，共 4 个 ROI；
- 所有 ROI 从原始像素裁剪，掩膜外背景固定为 RGB `(127,127,127)`；
- 三袋共用一个 SigLIP-L 冻结骨干和同一个低容量 gated 直接分类头；
- 每袋分别与六个基础视图组成 10 视图输入，最终对三袋 logits 取均值。

袋 0 使用与 E1 `matched_random_roi` 完全相同的种子与坐标。袋 1、2 只改变固定随机种子。不得按标签、来源、置信度、错误或实验结果选择袋。

## 3. 两个预注册变体

1. `multibag_mean`：三袋共享头，平均 logits 后计算二分类交叉熵，仅作机制消融。
2. `multibag_consistency`：在上述损失上固定加入权重 `0.10` 的三袋 Jensen-Shannon 一致性损失，为唯一主变体。

袋数、损失权重、阈值、随机种子和坐标规则在查看 F1 指标前冻结，不进行网格搜索。均值变体即使事后更高，也不得替代主变体作为确认结论。

## 4. 严禁输入

来源、批次、真实亚型、stage-1 概率、置信度、难度、预测正确性、错误类型或人工 ROI 均不得进入模型。亚型和来源只用于事后分层评估。

## 5. 评估

- 锁定五折 OOF；
- 三来源 source-LODO；
- 固定阈值 0.5；
- BAcc、AUC、敏感度、特异度、来源和六亚型分层；
- 5,000 次病例级配对 bootstrap；
- 同时对比锁定 C1、E1 `base_only`、E1 单随机袋以及 F1 均值消融。

## 6. 主变体晋级门槛

`multibag_consistency` 只有全部满足才 PASS：

1. 相对 E1 基础组，五折 OOF BAcc 至少 `+0.020`；
2. source-LODO BAcc 至少 `+0.015`；
3. 至少 2/3 留出来源为正增益，任一来源不得低于 `-0.020`；
4. OOF 和 LODO 均优于 E1 单随机袋；
5. B1、B2 的 LODO BAcc 各自下降不得低于 `-0.050`；
6. 计划质控满足：3 袋、随机 fallback 比例不高于 `5%`、组织覆盖率差均值不高于 `0.02`、P95 不高于 `0.05`、目标和袋内中心最小距离不低于 `0.15`、袋 0 坐标精确复现 E1。

未通过即关闭多随机袋一致性路线，不追加袋数、权重、种子、损失或阈值搜索。所有患者特征仅驻留服务器内存。
