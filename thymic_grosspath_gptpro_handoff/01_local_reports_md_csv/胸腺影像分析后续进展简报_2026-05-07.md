# 胸腺影像分析后续进展简报
日期：2026-05-07

## 1. 汇报目的

本简报用于承接《胸腺影像分析阶段进展与后续计划（2026-05-07）》之后的新增工作，集中说明团队在后续几轮实验中又完成了哪些关键内容、当前模型性能推进到了什么水平、以及下一步的判断依据。

当前所有结果继续统一使用以下口径：
- `160` 例病例、`172` 张图像、`12` 例多图病例
- 原始 `8` 类各 `20` 例，类别完全平衡
- 固定 `5-fold patient-level` 交叉验证
- 病例级主评估口径为 `case-level mean probability`

## 2. 新增完成的工作

上一版汇报之后，团队主要完成了四类新增工作。

第一，完成了 `Round3` 强表征模型接入。先后整理并接入了 `DINOv2`、`PLIP`、`BiomedCoOp` 和 `INTR` 等外部代码，其中最先跑通并给出明显正向结果的是 `DINOv2`。为此，我们新增了冻结特征提取与 `linear probe` 评估脚本，使强表征模型可以与现有 `registry + 5-fold split + case-level` 评估协议直接对齐。

第二，系统补齐了 `Task1-Task4` 在 `DINOv2` 主线下的首轮结果，并扩展测试了更大的 backbone 与多模型组合。除了 `vits14` 之外，又完成了 `vitb14`、`vitl14` 在关键任务上的对照，并进一步验证了 `Task3` 和 `Task4` 在新主线下最值得保留的结果带。

第三，完成了多条融合与轻量微调分支的去留判断。团队先后验证了 `PLIP` 单模、`DINO + PLIP stacking`、`class-wise alpha blend`、`MLP fusion head`、`DINO 轻量微调` 等分支。结果表明，这些方向大多没有超过当前最优的强表征单模或简单概率融合，因此已经基本可以从主线中降级。

第四，完成了当前最佳结果的正式收口。团队已经把 `Task1-Task4` 当前最值得汇报的配置、主指标、以及不同口径下的取舍整理成正式汇总表，便于后续用于导师汇报、论文表格和项目决策。

## 3. 新增关键结果

### 3.1 Task1 与 Task2

在强表征主线下，`Task1` 与 `Task2` 的结果已经比较可观。

- `Task1` 如果按主指标 `AUC` 汇报，当前最好是 `DINOv2 vitl14 + whole`：
  - `AUC = 0.8707`
  - `Balanced Accuracy = 0.7321`
- `Task1` 如果更强调类别平衡和特异性，当前更稳的是 `DINOv2 vits14 + whole`：
  - `AUC = 0.8161`
  - `Balanced Accuracy = 0.7607`
- `Task2` 当前最优是 `DINOv2 vits14 + whole`：
  - `AUC = 0.8638`
  - `Balanced Accuracy = 0.7875`

这说明胸腺大体图在二分类任务上已经可以被较强表征模型稳定利用，当前的主要瓶颈不再是“有没有信号”，而是“怎样把更难的任务做得更强”。

### 3.2 Task3：低危 vs 高危

`Task3` 相比前一阶段又有了实质推进。

当前最值得保留的正式结果是：
- `DINOv2 vitl14 + whole + fold-wise threshold`
- `AUC = 0.7621`
- `Balanced Accuracy = 0.7167`

这条结果比前一阶段旧主线更像样，也比 `vits14 whole` 更强，说明在风险分层任务上，更大的强表征模型和更合理的阈值决策是有效的。

### 3.3 Task4：WHO 五分类

`Task4` 仍然是当前阶段进步最明显的任务。

单模里，当前最强结果仍然来自：
- `DINOv2 vits14 + whole`
- `Macro-F1 = 0.5164`
- `Macro-AUC = 0.7413`

在此基础上，进一步补做的多模型融合结果显示：

- `DINOv2 vits14 + vitb14` 全局加权融合
  - `Macro-F1 = 0.5236`
  - `Macro-AUC = 0.7533`
  - 这是当前 `Task4` 最好的 `F1`

- `DINOv2 vits14 + vitb14 + PLIP` 三模型融合
  - `Macro-F1 = 0.5226`
  - `Macro-AUC = 0.7656`
  - 这是当前 `Task4` 最好的 `AUC`

这说明当前最有效的提升路径，不是继续在旧 `SuRImage` 结构里修修补补，而是利用强表征模型之间的互补性做更高质量的结果融合。

## 4. 已验证为负结果的方向

这段时间团队还完成了多条“看起来可能有帮助”的方向验证，但结果并不支持继续投入，主要包括：

- `PLIP` 单模替代 `DINO`
- `DINO + PLIP` 的 `stacking meta-head`
- `DINO + PLIP` 的 `class-wise alpha blend`
- `DINO + PLIP` 的 `MLP case-fusion head`
- `DINO vits14` 的 `head-only / last_block` 轻量微调
- `Task4` 的 `crop` 或 `whole_plus_crop` 继续升主线

这些实验的价值在于帮助我们正式排除低收益分支，避免后续继续在小幅波动和假提升上消耗时间。

## 5. 当前阶段判断

结合新增实验，当前可以形成三个比较稳定的判断。

第一，`Round3` 已经把项目主线从旧 `SuRImage` 小修，成功切换到了更强表征模型主线。当前模型真正明显变强，主要就是从这里开始的。

第二，`Task4` 已经形成了比旧主线明显更强的一组新结果。无论是单模 `DINOv2 vits14 whole`，还是后续的 `vits14 + vitb14` / `vits14 + vitb14 + PLIP` 融合，都已经稳定高于旧主线大约 `Macro-F1 ≈ 0.45` 的水平。

第三，便宜的后处理、小型融合头和轻量微调分支，基本已经摸到了当前阶段的上限。后续如果还想继续把模型做得“明显厉害”，需要进入更系统的结构性改进，而不是继续抠小数点。

## 6. 接下来建议

当前最合理的后续安排是：

1. 以 `DINOv2` 强表征为核心，正式确定 `Task3` 和 `Task4` 的主汇报口径。
2. 对 `Task4`，优先决定论文里究竟采用“最好的 `F1` 结果”还是“最好的 `AUC` 结果”作为主表。
3. 不再把主要时间投入到已判负的小分支上，而是把后续精力集中在更高一级的适配或更系统的融合方法上。

## 7. 一句话总结

如果用一句话概括上一版汇报之后的新增进展，那么当前最准确的总结是：

> 团队已经完成从旧 `SuRImage` 主线向强表征主线的切换，并在 `Task3` 和 `Task4` 上拿到了更强、更清晰、也更适合正式汇报的新结果，其中 `Task4` 已经从单模提升到多模型融合的更高水平。
