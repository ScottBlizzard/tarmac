# 模型结果与纯视觉审阅对照简报

日期：2026-05-09

## 1. 这份简报想回答什么问题

我们补做了一轮“纯视觉审阅”分析：不依赖正式训练好的分类模型，而是由大模型直接逐张查看胸腺大体图，只根据肉眼可见形态做盲猜，然后与正式模型结果做对照。

这份对照不是为了证明“大模型看图比训练模型更强”，而是为了回答三个更关键的问题：

- 大体图本身是否已经包含足够强的视觉信号；
- 模型最难的边界，是否也是人眼直观看图时最难的边界；
- 当前瓶颈更像是模型能力不足，还是图像本身就存在真实视觉重叠。

## 2. 两种结果的口径差异

| 项目 | 纯视觉审阅 | 正式模型结果 |
|---|---|---|
| 输入方式 | 逐张大体图直接看图盲猜 | 5-fold patient-level cross-validation |
| 是否训练 | 不训练，只看图判断 | 用标注数据训练分类器 |
| 结果层级 | 更接近 image-level 直觉判断 | case-level 正式评估 |
| 主要意义 | 看“图像本身是否容易看出规律” | 看“模型在正式协议下能做到什么水平” |

因此，两种结果不能完全按同一口径横向比较，但可以用来判断：模型学到的困难点，是否与纯视觉观察到的困难点一致。

## 3. Task5 与 Task6 的核心对照

Task5 定义为三分类：A/AB、B1/B2/B3、TC。

Task6 定义为六分类：A、AB、B1、B2、B3、TC。

| 任务 | 纯视觉审阅结果 | 当前最好模型结果 | 直接解读 |
|---|---|---|---|
| Task5 三分类 | 准确率 0.7045 | 准确率 0.6917；Macro-F1 0.6614；Macro-AUC 0.7959 | 三分类的视觉信号已经很强，纯看图和模型在同一量级 |
| Task6 六分类 | 准确率 0.5000 | 准确率 0.5167；Macro-F1 0.5127；Macro-AUC 0.8095 | 六分类明显更难，模型略优于纯视觉直观判断，但优势不大 |

这一组结果说明：

- 对于较粗的临床分组，图像本身已经能提供较强线索；
- 对于更细的六分类，训练模型确实比纯看图更好，但提升幅度仍然有限；
- 这提示我们：当前问题不是“完全看不出来”，而是“很多类别之间确实长得很接近”。

## 4. 纯视觉审阅中最容易和最困难的类别

### 4.1 Task5 三分类

| 类别 | 纯视觉审阅准确率 | 说明 |
|---|---:|---|
| A/AB | 0.9070 | 最容易识别，通常外观更淡、更平滑、侵袭感更弱 |
| B1/B2/B3 | 0.7031 | 中等难度，和 A/AB、TC 都有一定重叠 |
| TC | 0.3600 | 最难，仅凭大体图并不总是表现出“非常典型的癌样外观” |

Task5 的核心信息非常明确：最难的不是 A/AB 这一端，而是 TC 与 B 组之间的边界。

### 4.2 Task6 六分类

| 类别 | 纯视觉审阅准确率 | 说明 |
|---|---:|---|
| A | 0.3913 | 与 AB 边界高度重叠 |
| AB | 0.7000 | 六分类中相对最稳定 |
| B1 | 0.4348 | 常被拉向 B2 或低危组 |
| B2 | 0.6500 | 处于多类交界区，最容易成为“边界中心类” |
| B3 | 0.5238 | 与 B2、TC 均有重叠 |
| TC | 0.3600 | 仍然最难，常被看成 B2/B3 或甚至较低危类别 |

这里最关键的现象是：Task6 最困难的几个类，和正式模型最困难的几个类高度一致。

## 5. 模型最难的边界，与纯视觉最难边界是否一致

### 5.1 Task5

正式模型中，Task5 的主要短板是 TC 的召回率。

| 指标 | 当前最好模型 |
|---|---:|
| Task5 准确率 | 0.6917 |
| Task5 Macro-F1 | 0.6614 |
| TC Recall | 0.5000 |

模型最主要混淆方向是：

- TC -> B1-3：7 例
- B1-3 -> TC：5 例

纯视觉审阅最主要混淆方向则是：

- TC -> B123：10 例
- TC -> A_AB：6 例
- B123 -> A_AB：14 例

两边共同指出的核心问题是：TC 与 B 组之间存在显著视觉重叠。

### 5.2 Task6

正式模型当前最好结果：

| 指标 | 当前最好模型 |
|---|---:|
| Task6 准确率 | 0.5167 |
| Task6 Macro-F1 | 0.5127 |
| Task6 Macro-AUC | 0.8095 |

模型里最困难的类与边界：

| 项目 | 数值或现象 |
|---|---|
| A recall | 0.3000 |
| B2 recall | 0.3500 |
| 主要混淆对 | A -> AB，B1 -> B2，B2 -> B3，B2 -> TC，TC -> B2 |

纯视觉审阅里最明显的混淆对：

- A -> AB：10 例
- AB -> A：6 例
- B1 -> B2：7 例
- TC -> AB：6 例
- TC -> B2：6 例
- B3 -> TC：5 例

这说明：

- A 与 AB 的边界，本身就是连续谱；
- B1 与 B2 的边界，本身就不清；
- B2、B3、TC 三者之间共享较多深色、出血、异质性外观，但并不是简单线性递进关系。

## 6. 这份对照能说明什么

第一，纯视觉审阅并没有明显超过训练模型。

- 在 Task5 上，纯视觉准确率与模型准确率非常接近；
- 在 Task6 上，模型略高于纯视觉判断；
- 这说明模型确实学到了一些额外模式，但优势目前还不算特别大。

第二，模型最难的地方，并不是“模型乱学出来的”，而是图像本身就很难。

如果纯视觉和正式模型总是错在同一批边界上，那么更合理的解释是：

- 类别本身存在真实视觉重叠；
- 当前数据量不足以把这些边界拉开；
- 部分病例即使由人直观看图，也很容易犹豫。

第三，当前阶段最值得向医生追问的，不是“这些图是不是拍坏了”，而是：

- 这些高置信错例是否本来就处于临床边界区；
- gross image 与病理标签是否可能存在取材或对应关系偏差；
- 哪些病例即使由医生肉眼判断，也会存在明显犹豫。

## 7. 阶段性结论

这轮“模型结果 vs 纯视觉审阅”的对照，支持三个阶段判断：

1. Task5 三分类已经具备较强的视觉可分性，是一条临床上更自然、也更容易解释的主结果线。
2. Task6 六分类已经能做，但目前仍明显受限于类别边界重叠，尤其是 A/AB、B1/B2、B2/B3、B2/TC。
3. 当前瓶颈不是简单图像质量，而是数据量和边界病例数量不足。

因此，如果后续能补充更多病例，尤其是：

- TC 与 B2/B3 边界病例；
- A 与 AB 边界病例；
- B1 与 B2 边界病例；

那么对 Task5 和 Task6 的提升都很可能是实质性的，而不是只涨一两个点。
